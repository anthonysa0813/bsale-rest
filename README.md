# Bsale Store

- aplicación de tienda virtual donde podrás visualizar los mejores productos del mercado de bebidas.

## Tecnologías usadas para el desarrollo:

### Cliente:

- Javascript
- HTML
- CSS

### Servidor:

- Javascript
- NodeJs
- Express
- Dotenv
- Mysql2
- cors
